这是一个用于学习、追踪、筛选和综合 **wildfire**、**fire refuge**、**fire refugia**、**wildfire risk**、**evacuation**、**remote sensing** 和 **AI/GIS fire studies** 的 Codex skill。

它不是只服务博士开题，而是面向不同学习阶段的人：刚入门的学习者、课程作业阶段的学生、刚开始读文献的研究训练者、准备 proposal 的研究生，以及需要持续追踪前沿的进阶研究者。

这个项目把科学文献、数据集、开源项目、政策指南和研究趋势连接起来，覆盖 wildfire science、emergency management、fire ecology、remote sensing、geospatial AI、community resilience 等方向。

> 这个 skill 会区分 **fire refuge** 和 **fire refugia**：前者通常指人类安全和应急管理中的避难空间/避难策略，后者通常指生态学中的火后残存或低强度燃烧区域。

## 为什么做这个 Skill

Wildfire 研究横跨很多领域：fire ecology、disaster risk reduction、emergency management、remote sensing、GIScience、climate science、computer vision、public health 和 land-use planning。不同领域会使用不同术语，初学者很容易被概念、方法和文献系统分散。

这个 skill 的目标就是把这些线索串起来：

| 需求 | Skill 能做什么 |
| --- | --- |
| 入门学习 | 解释 wildfire、fire refuge、fire refugia、WUI、burn severity、evacuation、shelter-in-place 等核心概念 |
| 建立学习路径 | 把一个宽泛主题拆成适合入门、课程、文献阅读和研究训练的阅读队列 |
| 追踪新论文 | 检索最新论文、数据集、GitHub 项目、政策资料、会议页面和 Benchmark 页面 |
| 连接不同学科 | 串联生态、人类安全、遥感、AI/GIS、规划和治理视角 |
| 避免概念混淆 | 默认区分 human fire refuge 和 ecological fire refugia，除非用户明确要求综合分析 |
| 准备作业或 proposal | 生成结构化提纲、related work 表格、研究空白和项目 idea |
| 形成研究选题 | 把前沿趋势转化为可执行的研究设计和可投稿方向 |

## 研究范围

### Wildfire、风险与 Fire Refuge

| 方向 | 示例主题 |
| --- | --- |
| Wildfire hazard and risk | Fire likelihood、exposure、vulnerability、WUI risk、复合 drought-heat-fire risk |
| Fire behavior and spread | Fuel、wind、topography、spotting、fireline intensity、spread simulation、不确定性 |
| Fire weather and fuels | Fuel moisture、drought indices、heatwaves、lightning、fire danger rating |
| Human fire refuge | Community refuge、evacuation shelter、shelter-in-place、last-resort refuge、safer places |
| Evacuation and emergency management | 预警、交通网络、脆弱人群、决策支持、治理 |
| Planning and adaptation | Defensible space、fuel treatment、prescribed fire、cultural burning、land-use planning |

### Fire Refugia、生态与恢复

| 方向 | 示例主题 |
| --- | --- |
| Ecological fire refugia | Unburned patches、low-severity islands、moist gullies、riparian refugia |
| Biodiversity persistence | Habitat continuity、recolonization、species refuges、functional groups |
| Landscape resilience | Fire severity mosaics、patch configuration、repeated fire、climate refugia |
| Post-fire recovery | Vegetation recovery、erosion、hydrology、carbon、invasive species |
| Remote sensing of recovery | Burn severity、time-series monitoring、recovery trajectories |

### Remote Sensing 与 AI/GIS

| 方向 | 迁移到 Wildfire 研究的路径 |
| --- | --- |
| 卫星与航空影像 | 识别 active fire、burned area、burn severity、fuels、smoke 和 recovery |
| Foundation models | 将地理空间表征迁移到 wildfire mapping 和 monitoring |
| 时序建模 | 追踪 fire risk、fuel condition、burn severity 和 post-fire recovery |
| Graph and network models | 支持 evacuation、road vulnerability、infrastructure exposure 和 landscape connectivity |
| Physics-informed AI | 融合 ML 与 fire behavior、weather、hydrology 或 evacuation models |
| Uncertainty and robustness | 提升跨区域、跨季节、跨传感器和业务化应用的可靠性 |

### 好论文模式与科研表达

| 任务 | Skill 会提取什么 |
| --- | --- |
| 好论文阅读 | Central claim、evidence chain、figure/table roles、limitations、reusable structure |
| 图表设计 | Core conclusion、panel logic、visual job、uncertainty display、caption contract |
| 表格设计 | Dataset matrix、literature matrix、method comparison、policy comparison、research design matrix |
| Caption 写作 | Main message、data/method、geography/time、panel guide、uncertainty or boundary |
| 学术表达 | Abstract moves、introduction gap framing、related work grouping、results claim sentences、discussion boundaries |

## 评分标准

候选论文、数据集、政策资料和项目会按以下维度排序：

| 维度 | 含义 |
| --- | --- |
| 相关性 | 是否直接对应 wildfire、fire refuge、fire refugia 或用户的学习目标 |
| 清晰度 | 是否能帮助解释概念、方法、证据或领域结构 |
| 创新性 | 是否提出新问题、新方法、新数据集、新 Benchmark、新理论或新政策框架 |
| 证据强度 | 数据、验证、案例设计、不确定性处理和局限讨论是否扎实 |
| 可复现性 | 是否有代码、数据、协议、透明方法或可复用流程 |
| 实践价值 | 是否有助于规划、疏散、refuge 选址、风险降低或恢复 |
| 跨学科价值 | 是否连接生态、人类安全、遥感、AI/GIS、政策或社区韧性 |
| 用户匹配 | 是否匹配用户所处阶段：入门、课程、文献综述、proposal 或前沿追踪 |

## 注意事项

- 新论文、新数据集、GitHub 项目和政策页面变化很快，生成前沿报告时应使用联网检索。
- 中文报告中也建议保留英文论文标题和关键技术术语，方便后续检索。
- 如果涉及正在发生的火灾或生命安全问题，应以当地 emergency services 和官方 fire authority 指南为准。
- 在学术或政策写作中，使用 fire refuge 和 fire refugia 前应先明确概念定义。
