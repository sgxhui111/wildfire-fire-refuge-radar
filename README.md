# Wildfire Fire Refuge Radar

A Codex skill for tracking, screening, and synthesizing research frontiers on wildfire, fire refuge, fire refugia, WUI risk, evacuation, remote sensing, AI/GIS, and wildfire-related PhD proposal development.

## Skill Path

```text
skills/wildfire-fire-refuge-radar
```

## Use Cases

- Build daily or weekly wildfire research radar reports.
- Search and rank recent papers, datasets, GitHub projects, and policy sources.
- Compare fire refuge and fire refugia concepts.
- Prepare related work and PhD proposal directions.
- Generate publishable research ideas for wildfire, fire refuge, fire refugia, evacuation, WUI risk, or remote sensing/AI wildfire studies.

## Example Prompt

```text
Use wildfire-fire-refuge-radar to build a weekly research radar on wildfire, fire refuge, fire refugia, WUI evacuation, and remote sensing/AI methods.
```
